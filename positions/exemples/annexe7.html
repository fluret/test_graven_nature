Exemple d'utilisation de la position fixe dans un menu

Le code correspondant est :

En CSS :

        .page {
            width: 70%;
            border: 1px solid #000000;
            padding: 1em 1%;
            }
        .menu {
            position: absolute;
            top: 2em;
            right: 1em;
            background-color: #ffecce;
            border: 1px solid #666666;
            padding: 1em;
            width: 20%;
            }
        html>body .menu {
            position: fixed;
            }

En HTML :

        <body>
            <div class="page">
                ...
            </div>
            <div class="menu">
                <ul>
                    <li>lien_1</li>
                    <li>lien_2</li>
                    <li>lien_3</li>
                </ul>
            </div>
        </body>
