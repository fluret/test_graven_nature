Exemple d'utilisation de la position fixe pour une barre de menu fixée en haut de la fenêtre d'affichage

code CSS :

    .menufixe {
        position: absolute;
        top: 0;
        left: 1%;
        width: 98%;
        z-index: 2;
        text-align: center;
        background-color: #ffecce;
        border: 1px solid #666666;
        line-height: 2em;
        margin: 0;
        }
    html>body .menufixe {
        position: fixed
        }
    .content {
        z-index: 1;
        background-color: #f5f8fc;
        border: 3px solid #000000;
        margin: 0 2em;
        }

code HTML :

        <body>
            <div class="content">
                ...
            </div>
            <p class="menufixe">
                ...
            </p>
        </body>
