Exemple d'utilisation des flottants dans un menu

Dans les navigateurs conformes au standard CSS2, cette page se présente en deux colonnes: à gauche, le contenu principal, et à droite le menu. Le code correspondant est :

En CSS :

        .content {
            float: left;
            width: 70%;
            }
        .menu {
            margin-left: 80%;
            background-color: #ffecce;
            border: 1px solid #666666;
            padding: 1em;
            }

Et en HTML :

        <body>
        <div class="content">
        ...
        </div>
        <div class="menu">
        ...
        </div>
        </body>

Menu

Le contenu de cette zone est en position flottante.

Dans les navigateurs qui ne reproduisent pas les mises en page CSS, il s'affichera à la suite du contenu principal.
